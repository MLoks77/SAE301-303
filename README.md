<h1>SAE 301-303 : Sush'n Go</h1>

La sae 301-303 est un site de réservation à distances de boxes de sushis.
Sur ce site, l'utilisateur peut créer un compte afin de réserver jusqu'a 10 boxes de sushis.
Des fonctionnalités tels que : Un système de fidélité, les stats de l'entreprise sush'n Go, une carte interactive avec nos restaurants, sont disponibles pour l'utilisateur.

<h2>Languages</h2>

Cette SAE fut réalisé avec :

<ul>
  <li>HTML</li>
  <li>TAILWIND</li>
  <li>PHP & SQL</li>
  <li>Typescript</li>
</ul>

<h2>Comment lancer le projet</h2>

Pour voir le projet, importez la base de données disponible parmi les fichiers du repository et **n'oubliez pas de lancer ce projet dans un dossier htdocs avec XAMPP** par exemple.

Se positionner dans SAE301-303

```bash
cd .\frontend\sushngo\
```

```bash
ng serve
```

Rendez-vous ensuite sur http://localhost:4200/
